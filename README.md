# Data Warehouse and Analytics Project 🚀

Welcome to the **Data Warehouse and Analytics Project** repository!

This project demonstrates a complete end-to-end **Data Warehouse solution**, from raw data ingestion to analytics-ready datasets and reporting. It is designed as a portfolio project to showcase **Data Engineering + SQL + Analytics skills**.

---

## 🏗️ Data Architecture

This project follows the **Medallion Architecture (Bronze, Silver, Gold)**:

### 🔹 Bronze Layer
- Raw data ingested from source systems (ERP & CRM CSV files)
- Stored in SQL Server without any changes

### 🔹 Silver Layer
- Data cleaning and transformation
- Fix missing values, duplicates, formatting issues
- Standardization of data

### 🔹 Gold Layer
- Final business-ready data
- Star schema (Fact & Dimension tables)
- Used for analytics and reporting

---

## 📖 Project Workflow

### 1. Data Ingestion
- Load ERP and CRM CSV files into SQL Server

### 2. ETL Process
- Extract data from sources
- Transform and clean data
- Load into structured tables

### 3. Data Modeling
- Build Fact and Dimension tables
- Create Star Schema for analytics

### 4. Analytics & Reporting
- SQL-based analysis on business data
- Key insights:
  - Customer behavior
  - Product performance
  - Sales trends
